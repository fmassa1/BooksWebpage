# Books API 

This is a work-in-progress ASP.NET Core Web API for discussion boards for books and their chapters.

The project is mainly for learning and practicing .NET, Entity Framework Core, and REST API development.

---

## What It Does (So Far)

- Create books
- Retrieve books
- Add chapters
- Comment and reply to comments
- Store data using Entity Framework Core
- Test endpoints using Swagger


Planned (not finished yet):
- User Profiles
- Ratings
- Frontend Webpage

---

## Tech Stack

- ASP.NET Core
- Entity Framework Core
- SQL Database (via EF Core)
- Swagger

---

## Project Status

This project is not finished.

Some features only exist as models and are still being built.  
Expect missing endpoints and breaking changes.

---

## Running the Project

1. Clone the repository
2. Restore dependencies
3. Run the project using the .NET CLI
4. Open Swagger in your browser to test endpoints
